# site-novo

Site institucional da **Commex Tech** — protótipo estático, sem etapa de build. É só abrir `index.html` no navegador.

## Estrutura

- `*.html` — as 11 páginas (Home, Sobre, hub de Soluções, 7 módulos e Contato)
- `assets/css/style.css` — design system (cores, tipografia, componentes)
- `assets/js/main.js` — interações (menu, dropdown de Soluções, animações de entrada)
- `assets/img/` — capturas reais da plataforma

## Páginas

| Arquivo | Página |
|---|---|
| `index.html` | Home |
| `sobre.html` | Sobre nós (missão, visão, valores) |
| `solucoes.html` | Soluções (hub) |
| `cadastro-de-produtos.html` | Cadastro de Produtos (base da plataforma) |
| `ncm.html` | Classificação NCM com IA |
| `integracoes.html` | Integrações ERP |
| `duimp.html` · `lpco.html` · `du-e.html` · `nfe.html` | Módulos adicionais |
| `contato.html` | Contato |
