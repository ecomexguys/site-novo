Integração do ERP com o Portal Único.

O catálogo já existe no SAP, no TOTVS, no Oracle ou no OSGT. Mesmo assim, a cada SKU novo, alguém exporta tudo para planilha antes de classificar e declarar. As Integrações ERP são um módulo adicional que liga o sistema de origem ao Portal Único e deixa registro de cada operação no caminho.

O problema

O trabalho manual entre o ERP e o Siscomex.

Sem integração, todo SKU novo passa por três etapas de trabalho manual antes de chegar ao Siscomex.

1. Exportar a planilha do ERP

Alguém extrai o catálogo do SAP ou do TOTVS para um arquivo. Essa cópia fica fora de qualquer controle e, no dia seguinte, já não bate com o ERP.

2. Subir em uma ferramenta isolada

O arquivo entra em um sistema sem conexão com o ERP. Colunas fora de ordem, descrições truncadas e SKUs duplicados precisam de tratamento manual antes de a classificação começar.

3. Conciliar tudo de volta

O resultado retorna por planilha e alguém reconcilia linha por linha no ERP. As alterações feitas no meio do caminho não ficam registradas em lugar nenhum.

O que o módulo faz

Integração nativa com o ERP que a sua operação já usa

Conectores prontos para os principais ERPs do mercado, dois caminhos técnicos padronizados e um ambiente compatível com as exigências de segurança da sua TI.

Conectores SAP, TOTVS, Oracle e OSGT

Integração nativa com os ERPs mais comuns nas operações de comércio exterior. Sistema próprio entra pelo mesmo caminho, via API documentada.

API REST e SFTP com OAuth

API REST para integração em tempo real, SFTP para troca estruturada de arquivos. Autenticação OAuth nos dois caminhos, sem credencial trocada por e-mail.

Ambiente multi-tenant isolado, com audit log

Os dados de cada cliente ficam em ambiente próprio, separados dos demais. O audit log registra cada operação de integração: o que entrou, o que saiu, em que momento e por qual credencial.

Começar por SFTP e migrar para API

Dá para começar pela troca de arquivos via SFTP e migrar para API REST quando a agenda do seu time de TI permitir. A operação continua rodando durante a transição.
